# VISUAL_STYLE_GUIDE_v0.1

> Repository Draft  
> Project: **Space Sprout Sleuth (SSS)** & **Hunger, Harvest, & History (HHH)**  
> Status: **v0.1**

---

# 1. Purpose

This guide establishes the visual language shared by both curricula.

Primary goals:

- Professional educational appearance
- Scientific credibility
- Long-term maintainability
- Consistent classroom printing
- Shared identity across projects

---

# 2. Core Design Philosophy

The visual system combines:

- NASA mission documentation
- Scientific field notebooks
- Government technical manuals
- Clean modern educational worksheets
- Subtle in-universe styling

The visuals should never resemble a video game first.

The documents should appear authentic enough that a teacher could reasonably mistake them for professional instructional materials.

---

# 3. Locked Design Decisions

## Style

- Hybrid professional / in-universe
- Moderate immersion
- NASA-inspired mission documents
- Grayscale-first
- Diagrams before decoration
- Mixed layout system
- Blank bordered response boxes
- Phosphor icon family (locked)

---

# 4. Visual Tone

The documents should feel:

- Calm
- Clean
- Methodical
- Scientific
- Timeless

Avoid:

- Comic-book styling
- Cartoon decoration
- Excessive color
- Busy page layouts
- Heavy textures

---

# 5. Shared Identity

Both curricula should appear produced by related scientific organizations.

HHH uses the **Temporal Agricultural Archive (TAA)** insignia.

SSS will receive a matching **Space Agriculture Authority (SAA)** insignia designed with identical construction principles while remaining visually distinct.

---

# 6. Color Philosophy

Primary workflow:

- Grayscale first
- Accent colors only when instructional value exists

Accent colors are reserved for:

- warnings
- success
- scientific emphasis
- navigation

Everything else should remain printable in black and white.

---

# 7. Typography Principles

Typography should prioritize readability over personality.

Hierarchy:

1. Large mission title
2. Section heading
3. Subheading
4. Body text
5. Labels
6. Captions

No decorative display fonts.

---

# 8. Layout Principles

Use:

- generous whitespace
- consistent margins
- visible alignment
- modular sections

Panels should appear only when they improve comprehension.

---

# 9. Response Areas

Student writing spaces should be:

- rectangular
- bordered
- empty
- consistent sizing
- printer friendly

Never decorate answer boxes.

---

# 10. Diagrams

Diagrams always take precedence over decorative artwork.

Illustrations should exist only when they improve understanding.

---

# 11. Asset Reuse

Game assets may be reused selectively.

Educational documents should never feel like screenshots from a game.

Assets should function as scientific illustrations.
