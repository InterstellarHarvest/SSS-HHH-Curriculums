# VISUAL_STYLE_GUIDE_v0.3 — Decision Brief

> **Status:** Project-defining visual checkpoint  
> **Purpose:** Select the typography, grayscale foundation, institutional accent systems, and Phosphor icon treatment that will be locked into `VISUAL_STYLE_GUIDE_v0.3.md`.

## Repository Findings

The existing projects already provide a useful identity foundation:

- SSS strongly favors cyan and green interface accents.
- HHH and the TAA insignia strongly favor amber with cool cyan support.
- Both games use dark technical neutrals, but the curriculum must translate that identity onto white, grayscale-first pages.
- The game colors are generally too bright for unrestricted print-document use, so the decision lab uses darker, contrast-safe adaptations rather than copying the interface colors directly.

## Lead-Design Recommendation

### Typography

**IBM Plex Sans + IBM Plex Mono**

This pairing best supports the NASA-inspired mission-document direction while remaining readable in classroom materials, tables, diagrams, metadata, and answer keys.

### Shared Grayscale Foundation

**Cool Mission Gray**

This gives both curricula a restrained technical character without requiring visible blue color. It also leaves enough room for TAA’s amber identity to feel archival without making the entire shared system warm.

### SSS Accent

**Orbital Cyan + Botanical Green**

This is the clearest professional continuation of the existing SSS identity. Cyan identifies the scientific and space-operational layer; green is reserved for botanical or successful-growth emphasis.

### HHH Accent

**Archive Amber + Record Cyan**

This directly reflects the TAA insignia and HHH interface. Amber identifies archive, chronology, and preserved evidence; cyan supplies the shared future-science connection.

### Phosphor Icon Treatment

**Regular icons at a reinforced print weight**

The proposed medium simulation sits between ordinary Regular and Bold. It remains clean on screen but survives classroom printing and photocopying better than a light stroke. The final guide can implement this by choosing the closest published Phosphor weight or by specifying size-dependent weight rules.

## Decisions Required Before v0.3 Is Locked

- Typography pairing
- Shared grayscale foundation
- SSS accent palette
- HHH accent palette
- Default Phosphor icon treatment

Use `VISUAL_STYLE_GUIDE_v0.3_DECISION_LAB.html` to compare the choices in both SSS and HHH page contexts and in grayscale.
