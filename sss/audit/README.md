# SSS Curriculum Audit

This directory holds the curriculum and content audit for **Space Sprout Sleuth (SSS)**.

## Files

- **`SSS_MASTER_AUDIT_v0.1.md`** — the human-readable curriculum/content audit.
- **`data/SSS_STATIC_CONTENT_INVENTORY_v0.1.json`** — machine-readable supporting inventory data that backs the audit.

## Versioning

- **v0.1 is preliminary.** It will be superseded by later audit versions.
- Older versions should be **retained for traceability** until the audit reaches **v1.0**.

## Canonical source

The canonical game source is the **`main` branch of Space Sprout Sleuth**.
